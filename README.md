# InterviewAI 🚀

An AI-powered interview assistant that simulates real interviewer feedback by analyzing coding solutions and providing structured insights.

---

## 📌 Overview

InterviewAI helps candidates improve their problem-solving skills by giving **interviewer-style feedback** on coding solutions. Instead of just showing the correct answer, it focuses on **how to improve your approach**.

---

## ✨ Features

- 🧠 Analyze coding solutions using AI
- 📊 Provides structured feedback:
  - Approach analysis
  - Mistake identification
  - Optimization suggestions
  - Time & space complexity
- 💬 Simulates real interview feedback
- ⚡ Fast and simple UI for quick practice

---

## 🏗️ Tech Stack

- **Backend:** Python, Flask  
- **AI Engine:** OpenAI API  
- **Frontend:** HTML, JavaScript  
- **Environment:** python-dotenv  

---

## ⚙️ How It Works

1. User enters a coding problem and solution  
2. Backend sends the data to the AI model  
3. AI analyzes the solution like an interviewer  
4. Structured feedback is returned and displayed
